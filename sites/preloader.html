Preloader

At first you need to make your html file and add bascic code

<div class="preloader">
    <div class="loader">
        <div class="loader-inner"></div>
    </div>
    <h3>Loading website</h3>
</div>

<div class="content none">
    Blabla
</div>

Next you need to find or make your own a loader, for better performance make it in CSS or use SVG file. These are needed lines of code in CSS. And inside class loader add your own.

body {
    margin: 0;
    padding: 0;            
}
        
.preloader {
    position: fixed;                
    background-color: white;
    width: 100vw;
    height: 100vh;
    display: flex;
    justify-content: center;
    align-items: center;
    margin: 0;
    padding: 0;
    flex-direction: column;
    transition: opacity 0.5s ease;
}

.preloader-finish {
    opacity: 0;
    pointer-events: none;
}

.loader {
    // You can add your own properties here
    width: 30px;
    height: 30px;
    z-index:3;
    border: 4px solid #566b78;
    animation: loader 2s infinite ease;
}

.loader-inner {
    vertical-align: top;
    width: 100%;
    background-color: #566b78;
    animation: loader-inner 2s infinite ease-in;
}

.none {
    display: none;
}

@keyframes loader {
    0% {
        transform: rotate(0deg);
    }  
    25% {
        transform: rotate(180deg);
    }  
    50% {
        transform: rotate(180deg);
    }  
    75% {
        transform: rotate(360deg);
    }  
    100% {
        transform: rotate(360deg);
    }
}

@keyframes loader-inner {
    0% {
        height: 0%;
    }  
    25% {
        height: 0%;
    }  
    50% {
        height: 100%;
    }  
    75% {
        height: 100%;
    }  
    100% {
        height: 0%;
    }
}

And in the end you should add these simple javascript. To the site. It uses the load eventlistener that checks when site is loaded and then it perform the function inside. 

window.addEventListener(
    'load', 
    () => {
        const preloader = document.querySelector('.preloader')
        const section = document.querySelector('.content')
        preloader.classList.add('preloader-finish')
        section.classList.remove('none')
    }
)
